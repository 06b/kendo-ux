WebComponentsIcons

  • Icon modificator - situated at the bottom right corner of the icon.
  • Smaller/larger size of common icons to avoid their usage with diffetent font size. Add more details to the larger icon, if needed. Use suffix -sm or -lg.
  • Generic icons: the name should describe the illustrated object.
  • Specific icons: the name should describe the functionality icons are used for.
  • CSS class naming convention - icon could be associated to more than one class name (one generic class and one specifically related to the usage of the icon).
